# Kinohub Search Helper

Небольшое расширение добавляет кнопку «Смотреть на KinoHub» в контекстное меню и на страницы популярных сервисов: Кинопоиск, IMDb, Rotten Tomatoes, Metacritic, MyAnimeList и Shikimori.

![Screenshot](screenshot.png)

![Screenshot](screenshot2.png)

Как установить? 
1. Нажмите зеленую кнопку «Code», а затем выберите «Download ZIP».
2. Распакуйте скачанный файл в любую удобную папку на компьютере.
3. Откройте страницу расширений в браузере, включите «Режим разработчика» (Developer mode) и нажмите кнопку «Загрузить распакованное» (Load unpacked), выбрав папку расширения.
