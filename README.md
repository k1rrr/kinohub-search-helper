# Kinohub Search Helper

Расширение для удобного перехода с сайта Кинопоиск на Kinohub.

### Основные функции:
* **Кнопка "Смотреть":** Добавляет кнопку "Смотреть на KinoHub" прямо на страницу фильма или сериала на Кинопоиске.
* **Быстрый переход:** Экономит время, исключая ручной поиск контента на платформе.
* **Нативный дизайн:** Кнопка аккуратно вписывается в интерфейс Кинопоиска.
